Used Libraries: turtle, keyboard, random, time, json, copy
Starting the programm you are given
three options: Play, Train AI, Let AI play from Trainingdata
Play:
    You can play FlappyBird yourself
Train AI:
    You are asked about the parameters
    to use for training the AI
    Then the AI starts training
    After the Training is done
    you can name the file where
    to save the trainingdata
    containing the "brain" of
    the AI
Let AI play from Trainingdata:
    You give the AI the name
    of the file with the
    Trainingdata and it plays
    FlappyBird by itself
